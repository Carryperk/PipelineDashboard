subject Overview

Here are the commands available to you to update your DashboardHub environment from your local machine or from your Continuous Integration environment.

Start Build This command will show that a build has been started and initiate the timer Finish Build This command will show that the build is complete Build Failed This command will show that the build is complete Start Deploy This command will show the deployment of the release has started Finish Deploy This command will show the deployment of the release has completed and the timer will stop Deploy Failed This command will show the deployment of the release has completed and the timer will stop
